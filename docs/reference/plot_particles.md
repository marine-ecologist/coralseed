# Plot output from settle_particles

Function to plot output of seed_particles in base R for quick cheks

## Usage

``` r
plot_particles(
  input = NULL,
  seascape = NULL,
  parameter = "dispersaltime",
  pch = 16,
  size = 0.5,
  ...
)
```

## Arguments

- input:

  input

- seascape:

  seascape habitat input

- parameter:

  parameter (column from coralseed)

- pch:

  pch

- size:

  size

- ...:

  passes functions
